# Auctioneer

A very simple auction app where users can create and participate in real-time auctions.

## Tech Stack

- Mongo
- Node
- Express
- Socket.io

## How to use

- Clone the repo ```git clone https://github.com/SyedZeeshanHasmi/Auctioneer.git```
- ```cd Auctioneer/```
- ```npm install```
- ```npm start```
- App is now running in ```localhost:3000```
